# Quickstart

## Run

```bash
pnpm dev
# or: npm run dev / yarn dev / bun dev
```

Open `http://localhost:3000`.

## Use

- Add a task: open the modal, enter text, answer urgent/important.
- View by quadrant: tasks auto-place into Q1–Q4.
- Focus Mode: toggle to view all Q1 or one-at-a-time.
- Complete: mark as done → confetti → appears in Completed view.
- Manage completed: delete any completed task.

## Preferences & Motion

- Animations are subtle and respect reduced-motion.
- Toggle reduced-motion and sound in settings (planned).

## Data

- Saved locally (IndexedDB), with localStorage fallback.
- Import/Export JSON available from settings (planned).
